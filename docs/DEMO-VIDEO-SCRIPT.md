# Marcus — 45-Second Demo Video Script

Record a screen capture on macOS. No intro slide needed. Start with the browser already open.

Total runtime: 45 seconds. Publish at 1080p minimum.

---

## Shot 1 (0–8s): Sign in

**On screen:** Browser at https://marcus-second-brain.com. Click "Add to Claude". GitHub OAuth consent screen appears. Click "Authorize". Redirect back to claude.ai with Marcus connected.

**Voiceover:** "Connect Marcus to Claude with one click — GitHub OAuth, no password stored."

---

## Shot 2 (9–18s): Save a note

**On screen:** Claude chat open. Type and send: "Save a note about Atomic Habits — habits compound over time. Tags: books, productivity." Watch the tool call appear and the confirmation response. Cut to the GitHub repo in a browser tab showing the newly created `20-topics/atomic-habits.md` file.

**Voiceover:** "Marcus writes the note directly to your private GitHub repo. You own every file."

---

## Shot 3 (19–27s): Search the vault

**On screen:** Back in Claude chat. Type and send: "Search my vault for anything about Cloudflare Workers." Watch the search results return with a note summary.

**Voiceover:** "Search retrieves the right note from hundreds — by keyword, tag, or topic."

---

## Shot 4 (28–36s): Recall a memory

**On screen:** Claude chat. Type and send: "Recall my preferences." Watch the `recall` tool call and the response showing stored preferences.

**Voiceover:** "Preferences and context you've stored once are available in every future session."

---

## Shot 5 (37–45s): View vault on GitHub

**On screen:** Switch to the GitHub repo tab. Scroll through the folder tree: `00-daily/`, `20-topics/`, `40-projects/`, `15-memory/`. Click into one file to show the Markdown content with YAML frontmatter.

**Voiceover:** "Plain Markdown files, standard Git history, yours forever."

---

## Recording Notes

- Disable notifications before recording (macOS: Focus → Do Not Disturb).
- Use a clean browser profile with only the Marcus OAuth tab and Claude open.
- Keep the demo account's vault small (use the seeded notes from `seed-reviewer-vault.ts`) so search results are uncluttered.
- Export at 1920×1080, H.264, ≤100 MB for upload to YouTube or Loom.
- After recording: paste the public URL into `docs/SUBMISSION-ANTHROPIC.md` and `docs/SUBMISSION-OPENAI.md` under "Demo Video".
